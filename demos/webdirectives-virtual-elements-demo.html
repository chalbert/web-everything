Web Directives: Virtual Elements

A control:for-each / control:if directive expressed as a real HTML comment marker — upgraded by the customComments registry (#1131), its options parsed by the default namespaced parser (#1132). The markers are virtual elements: they carry logic but add no node to the layout tree, so the CSS Grid below keeps its direct parent–child relationship intact. Spec: webdirectives.

Layout preservation

The middle card is gated by <!-- control:if when="showMiddle" -->. Toggle it: the grid never reflows around a wrapper because the marker is a comment, not an element.

List rendering (no wrapper node)

<!-- control:for-each items="members" --> renders one row per item between the opening and closing markers. The <ul>'s only element children are the <li> rows; the directive itself contributes comment nodes only.
