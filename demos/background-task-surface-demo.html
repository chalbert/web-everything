Background Task Surface Playground

A live test surface for the Background Task Surface block — the <background-tasks> element that hosts long-running work off-view after it goes async. Each card drives a real surface through a scripted Loader handle and a badge proves an invariant of the contract: a task is registered via a bubbling background-task-register event, the surface subscribes to the Loader's own state machine, and announces completion/failure in a polite role="status" region.

The scripted handle is the test twin of a real Loader at escalation:async — the surface only needs subscribe/getSnapshot/retry off-view. The scenarios and assertions come from the same shared fixture module the unit tests use, so the badges and CI can't drift. This is the route-only baseline; the reload-durable tier is the separate #134 adapter.
