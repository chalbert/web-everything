The runnable proof of the changelog-manifest contract (#1057): a per-release manifest of per-module change entries — severity, type, and a migration ref present iff a change is breaking and mechanically applicable.

The contract is type-only; the runtime provider/reader + swap registry are impl in Frontier UI. So this demo supplies its own in-demo implementation — honest for a browser demo — to prove the contract is realizable. The section below asserts each contract invariant live.
