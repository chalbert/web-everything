Module-as-a-Service Playground

A live test surface for backlog #081 — a provider that serves one authored component in any form, chosen by a single request param. Pick a form per card: the same <component> definition is returned as declarative source, a Web Component class, template HTML, or JSX — and the live element below is registered from the served WC form so you can see it run.

This is the v1 walking skeleton: a single synchronous resolve. The resolver owns no transform of its own — it dispatches to the same shared transforms the rendering adapters demos use, so the served form provably matches the documented one (no drift). Reactivity, transpileTarget, and ESM-over-HTTP delivery are phase 2 — requesting an unsupported param is flagged, never silently dropped.
