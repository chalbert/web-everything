Web Contexts — declarative contexts & claim-based lookup

Exercises the Web Contexts standard in a real browser: a declarative <script type="context"> registration, hierarchical resolution through the injector tree, and the strict vs. flexible lookup modes (#1117). A child and a parent scope each hold a context of the same type; the child's context only claim()s user.* queries.

Toggle the lookup mode and the query below. strict always resolves to the closest context (the child), ignoring claim(). flexible (the Most-Flexible-Default, #911) consults claim() and walks up the chain when a context declines — so a query the child declines defers to the claiming parent.
