Keyboard Shortcuts Demo

The runnable proof of the keyboard-shortcuts block (#1139): a chord-normalization module folds a raw KeyboardEvent into one canonical chord — folding the OS accelerator (Cmd on macOS, Ctrl elsewhere) onto a single Mod token — and a declarative shortcut registry fires the matching entry regardless of where focus sits, including across shadow boundaries.

This demo supplies its own in-demo implementation of the two pieces the block names — honest for a browser demo — to prove the block is realizable and graduate it from draft to active. The section below asserts each contract invariant live; the live registry below it lets you try the chords on the real page.
