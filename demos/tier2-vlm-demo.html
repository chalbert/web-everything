Tier-2 In-Browser VLM — Florence-2 via Transformers.js

Runs the #1082 transformers-vlm provider entirely in your browser on WebGPU (ONNX Runtime Web) and renders the #1080 rich envelope (description / tags / regions overlay) for a chosen screenshot. Only the model outputs cross the design-refs provider seam — never the model (#475 no-leakage).

Opt-in, device-gated. This page downloads a ~2 GB Florence-2 model and requires a WebGPU-capable desktop browser plus @huggingface/transformers installed (npm i @huggingface/transformers). It never runs by default or on mobile — pick a screenshot and press Analyze to opt in. CI/batch can’t run it (hence #1142 is split from the verified provider core #1082, whose pure mappers are unit-tested without the model).
