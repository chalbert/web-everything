Web Graph Conformance Demo

The runnable proof of the Web Graph contract (#1443): a semantic GraphSpec in, a deterministic layered-DAG layout invents coordinates, and an SVG renderer draws themed and accessible output. Scored on the three conformance axes — semantic fidelity, theme application, and accessibility.

The contract is type-only (graphs/contract.ts); the native-first layered-DAG layout (#1444), the SVG renderer (#1445), every adapter, and the swap registries are impl and live in Frontier UI. So this demo supplies its own in-demo layout + SVG renderer — honest for a browser demo — to prove the contract is realizable.
