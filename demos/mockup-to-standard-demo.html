Mockup-to-Standard Playground

A live surface for backlog #086 — a design-to-code front door on the #094 upgrader pipeline. Instead of lifting existing code, each card lifts a UI mockup (static image, Figma frame, interactive prototype) through the same engine and the same verify gate: mockup → neutral structure (IR) → generated <component> → verify gate → live element.

AI is a swappable, no-leakage provider. Mockup analysis is a CustomVisionProvider behind the registry. Here a deterministic, keyless reference provider stands in until the Plateau vision service (#475) ships; only the neutral structure crosses back into the standard. The generated <component> is the same declarative form the adapters consume — no parallel generator.
