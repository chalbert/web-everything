Loader → Background Task Handoff Playground

The producer half of the Background Task Surface handoff: a real Resource Loader running backgroundLoad() escalates a slow load into a live <background-tasks> rail. Each card drives the real pair — not a scripted mock — and a badge proves an invariant of the escalation contract.

Escalation:async trigger. A load escalates exactly when it crosses the Loader's debounce threshold into loading — the moment the Loader already decided the work was slow enough to surface a pending UI. Fast loads that resolve first never enter loading, so they never reach the rail. The threshold is shortened here for a snappy demo. Scenarios and the controllable-promise lever come from the same shared fixture module the unit suite imports, so the badges and CI can't drift. This wires #152.
