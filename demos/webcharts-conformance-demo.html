The runnable proof of the CustomChartRenderer contract (#1334): a semantic Vega-Lite L1 ChartSpec plus resolved Web Theme tokens in, themed and accessible SVG out. Scored on the three conformance axes — semantic fidelity, theme application, and accessibility.

So this demo supplies its own in-demo SVG renderer — honest for a browser demo — to prove the contract is realizable. The section below scores each axis live against the src/cases/webcharts/* cases.
