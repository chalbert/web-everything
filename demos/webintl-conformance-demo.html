Web Intl Conformance Demo

The runnable proof of the CustomIntlProvider contract (#1054): one provider hands back Intl.Collator / DateTimeFormat / NumberFormat / RelativeTimeFormat for a requested locale, so an app gets locale-aware collation, date, number, and relative-time formatting through one swappable seam.

The contract is type-only; the runtime provider/reader + swap registry are impl in Frontier UI. So this demo supplies its own in-demo implementation — honest for a browser demo — to prove the contract is realizable. The section below asserts each contract invariant live.
