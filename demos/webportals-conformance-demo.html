Web Portals Conformance Demo

The runnable proof of the Web Portals runtime contract (#1001) in a real browser: a deeply nested component portals a modal to a top-level outlet that escapes its stacking context, while context preservation (resolved via logical ancestry) and logical event bubbling (a click inside the portal firing a handler on the logical ancestor via composedLogical) both verifiably hold.

This exercises the actual webportals plug in the browser — the directive projects the modal into the outlet, logicalParent links the projection to its declaration site, and dispatchLogical walks the logical chain. The section below asserts each contract invariant live.
