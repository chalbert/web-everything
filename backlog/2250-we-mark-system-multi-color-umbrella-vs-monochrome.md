# WE mark system — multi-color umbrella vs monochrome

The user-favored WE direction is the **constellation-color Venn** (WE-indigo + FUI-cyan + Plateau-violet
overlapping = "Web Everything = the whole constellation unified"). This breaks the ratified-to-be
"one gradient + white glyph per project" construction rule. Decide the system: **multi-color umbrella
hierarchy** (WE = all three colors; FUI/Plateau = their own single color) vs **strict monochrome**
(keep the rule; WE stays single-hue — the v6 W or a mono Venn). Amendment to #2209's construction-system
fork. Candidates on `plateau:branding.html#journeys` (Round 8: WE-37/38).
