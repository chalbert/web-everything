# Roll the HTML/JSX autoToggle onto more block pages and graduate the adapter status

The build-time `autoToggle` (author HTML once, JSX pane generated by the `htmlToJsx` filter) is live on the for-each block page; dropdown stays on the manual toggle because its Altitude-1 example is annotated pseudo-HTML (comments inside a start tag) that won't parse.

Roll `autoToggle` onto the remaining block pages whose examples are clean, parseable HTML, and once coverage is meaningful, graduate `jsx-adapter` in `src/_data/adapters.json` from `draft`. Constraint: `autoToggle` requires parseable HTML; annotated/pseudo snippets keep the manual `sourceToggle` macros. Parallels the `<component>` adapter's `component-converter-playground` track.
