# Design-ref phase 3: vision codification of the corpus into intents

The codification phase: run a vision pass over the corpus to fill the per-screenshot tags deferred in phase 1 (surface, refined register, layout, notable patterns), then harvest recurring UI paradigms into candidate composable intents/blocks for Web Everything. This is where the collection pays off — reference screenshots become standards input.
