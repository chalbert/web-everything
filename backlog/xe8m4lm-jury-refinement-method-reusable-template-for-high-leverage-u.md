# Jury-refinement method — reusable template for high-leverage UI/design decisions

Codify the method proven on the icon grammar (artifact 66248282) as a reusable template. Loop: parallel multi-agent JURY (lenses: usability/visual/a11y/design-systems, plus IA/ops for structure calls) -> EXPOSE FORKS (never average away disagreement; develop each fork via an advocate) -> per-option RATINGS 1-5 (not just a single pick) -> WEAK-FLAG when the best option is below threshold -> SEARCH new candidates (keep the old) -> LIVE TRUTH-CHECKS on the surface (collision/duplicate detection synced to current picks, distinct state-vs-action check) -> HUMAN-GATE (jury proposes, human disposes; no self-ruling).
