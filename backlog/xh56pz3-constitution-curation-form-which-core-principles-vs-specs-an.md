# Constitution curation + form — which core principles vs specs, and derived-index vs artifact

core principles, the specifics become *specs* (§3b split), no duplication — but deferred the **form**. This
item decides (1) **which** principles are constitution vs specs (the curation), and (2) the **form**: (a) a
derived index of statute anchors, or (b) a standalone artifact — (b) back in play now that a non-duplicating
split defuses its drift objection. NOT prepared — needs the prep pass.

invariant explicit, and the injection mechanism named — so the [#2561] F4 mechanism is fully realized. First:
