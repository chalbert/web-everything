Reviewing all branding across the 3 repos (2026-07-03, plateau-app `branding.html` / `npm run gen:branding`), the user's clear favorite mark is the **Plateau favicon** (layered mesa silhouette, violet→cyan gradient) — confirming my ranking: real symbol/metaphor beats letterform marks (WE "W+ghost-E", FUI "FUI"), and 16px legibility matters.

**Why:** it's the only mark with an actual idea (a literal plateau) rather than initials; depth via opacity layers; survives favicon size.

**How to apply:** when creating or revising marks/icons for these projects, aim for symbol-first, 16px-legible designs in the shared construction system (40×40 squircle, rx12, gradient, white glyph). Known open branding issues: FUI logo weakest; WE ghost-E muddy at 16px; WE's `plateau.svg` (amber) and `frontierui.svg` ("FU") disagree with those products' own marks; Plateau has no full logo asset yet.
